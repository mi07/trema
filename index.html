Trema

OpenFlow Framework for Ruby/C

  • You can download this project in either zip or tar formats. Also you can clone the project with Git.

    Get Trema Today!

  • Then run build.rb and straight away you have a working Trema installation. 

    $ ./trema/build.rb

  • To start using Trema, cbench is a good entry point. cbench is a program for testing OpenFlow controllers by generating packet-in events for new flows. To estimate the performance of Trema, run the following command: 

    $ ./build.rb cbench

  • Some of OpenFlow controller applications developed on top of Trema can be found on Trema Apps. Trema Apps is not main part of Trema framework but you can refer to them as examples or references.
