Trema

Full-Stack OpenFlow Framework for Ruby/C

  • Trema is a full-stack, easy-to-use framework for developing OpenFlow controllers in Ruby/C.

    Get Trema Today!

  • Setup is very easy. Run build.rb in the Trema distribution and straight away you have a working Trema installation. 

    $ ./build.rb

  • Write your own OpenFlow controller by adding message handlers to your controller class just like Rails. 

    # Controller class
class MyController < Controller
  # packet_in handler
  def packet_in dpid, msg
    send_flow_mod_add(
      dpid,
      :match=>ExactMatch.from(msg),
      :buffer_id=>msg.buffer_id,
      :actions=>ActionOutput.new(msg.in_port+1)
    )
  end
end

  • Run the controller you wrote with trema run command. You can describe the network topology in which the controller is executed using network DSL and pass it to trema run. 

    $ cat network.conf
vswitch { dpid "0xabc" }
vhost "host1"
vhost "host2"
link "0xabc", "host1"
link "0xabc", "host2"

$ ./trema run mycontroller.rb -c network.conf

  • Some OpenFlow controllers developed on top of Trema can be found on Trema Apps repository. You can refer to them as examples or references.
